    JMeter 资源合集

    Apache JMeter 是一个 Java 语言编写的开源性能测试工具，是目前最主流的开源性能测试工具。本资源集源自 awesome-jmeter，MeterSphere 团队针对国内用户进行了汉化、优化和进一步整理。
